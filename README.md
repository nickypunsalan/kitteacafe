# Kittea Cafe

An on-going personal project to demonstrate use of the MERN stack.

## Things you'll need

* [npm](https://www.npmjs.com/)
* [Node.js](https://nodejs.org)

## Installation

1. Clone the repo

        git clone https://github.com/nickypunsalan/kitteacafe.git

2. Install NPM packages

        npm install

3. Usage

        npm run start
